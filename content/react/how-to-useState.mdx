<Text>A basic example of how to use useState. Below is a live playground where you can edit the code. Try changing the count increment</Text>


```jsx react-live
function useStateExample() {
  // Declare a new state variable, which we'll call "count"
  const [count, setCount] = React.useState(0);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
}
```
